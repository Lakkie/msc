Mapping‌ (or ‌function‌) is a fundamental concept in mathematics that establishes a precise relationship between elements of two sets. Formally, a mapping f from a set A (the ‌domain‌) to a set B (the ‌codomain‌) assigns to each element x in A exactly one element f(x) in B, often written as f:A→B.
映射‌ (又称函数‌) 是数学中的一个基本概念，它在两个集合的元素之间建立了精确的关系。从形式上讲，从集合A（定义域）到集合B（‌值域）的一个映射f，为A中的每个元素x分配一个B中的元素f(x)，通常写成f:A→B.
‌
Injective 单射：不同输入映射到不同输出。
‌Surjective 满射：B 中每个元素至少被一个 A 的元素映射到。
‌Bijective 双射：既是单射又是满射，存在一一对应关系。
